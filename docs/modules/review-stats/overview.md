# 借用评价 + 数据统计

## 模块概述

增加借用评价机制和数据统计功能，提升平台信任度和数据透明度。

## 目标用户

- 车位业主（查看评价和统计数据）
- 借用方（评价车位体验）
- 管理员（查看平台统计）

## 功能清单

### 1. 借用评价机制

| 功能 | 说明 |
|------|------|
| 借用结束评价 | 借用结束后，借用方可对车位评价（1-5星） |
| 评价内容 | 评分 + 可选文字评价（如"车位好停"、"位置偏"） |
| 评价展示 | 车位详情页显示历史评价和平均分 |
| 业主查看 | 业主可在"我的车位"查看收到的评价 |

**评价流程**：
```
借用结束 → 弹窗评价 → 提交 → 保存到数据库 → 更新车位平均分
```

### 2. 数据统计功能

| 统计项 | 说明 | 展示位置 |
|--------|------|----------|
| 车位总览 | 总车位数、已发布数、可用数 | 管理后台 |
| 借用统计 | 总借用次数、完成率 | 管理后台 |
| 用户统计 | 总用户数、活跃用户数 | 管理后台 |
| 收入统计 | 总交易额（估算） | 管理后台 |
| 车位热度 | 被借用次数最多的车位 Top 10 | 管理后台 |

**统计维度**：
- 时间维度：今日、本周、本月、全部
- 区域维度：A~G 区分别统计

## 数据模型

### 新增表：reviews

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| borrow_id | INTEGER | 关联借用记录 |
| spot_id | INTEGER | 关联车位 |
| reviewer_id | INTEGER | 评价人（借用方） |
| rating | INTEGER | 评分（1-5） |
| comment | TEXT | 评价内容 |
| created_at | DATETIME | 创建时间 |

### 新增字段：spots

| 字段 | 类型 | 说明 |
|------|------|------|
| avg_rating | REAL | 平均评分 |
| review_count | INTEGER | 评价数量 |

## 技术实现

### 后端

1. **评价 API**：
   - `POST /borrows/:id/review` - 提交评价
   - `GET /spots/:id/reviews` - 获取车位评价

2. **统计 API**：
   - `GET /admin/stats` - 增强现有统计
   - `GET /admin/stats/hot-spots` - 热门车位

### 前端

1. **评价弹窗**：
   - 借用结束后自动弹出
   - 星级选择 + 文字输入

2. **评价展示**：
   - 车位详情页显示平均分 + 历史评价
   - "我的车位"显示收到的评价

3. **统计页面**：
   - 管理后台增加统计图表

## 验收标准

- [ ] 借用结束后弹出评价弹窗
- [ ] 可提交 1-5 星评分 + 文字评价
- [ ] 车位详情页显示平均评分和历史评价
- [ ] 管理后台显示完整的统计数据
- [ ] 统计数据实时更新
