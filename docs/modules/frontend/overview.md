# 前端模块 概要设计

## 目标
提供 ParkingHub 的用户界面，包括登录、首页、发布、借用、记录等功能页面。

## 范围
### 包含
- 登录页（手机号 + 验证码）
- 完善信息页（昵称 + 楼栋 + 单元 + 区域）
- 首页（附近可借车位 + 我的车位 + 最近借用）
- 发布共享页
- 车位详情页
- 借用确认页（含一键通知管家）
- 借用记录页

### 不包含
- 管理后台（后续单独做）
- 微信小程序（后续考虑）

## 技术方案
- HTML + Tailwind CSS + Vanilla JS
- SPA（单页应用，hash 路由）
- 零 npm 依赖
- 静态文件由 Nginx 直接提供

## 依赖模块
- 所有后端 API（auth、spot、borrow、building-zone）
- localStorage 存储 JWT token

## 核心原则
- 移动优先
- 30 秒原则（每个操作 ≤ 30 秒）
- 简洁直观
