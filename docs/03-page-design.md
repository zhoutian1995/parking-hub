# 页面设计

## 页面总览

| 页面 | 用途 | 谁用 |
|------|------|------|
| 登录页 | 手机号登录/注册 | 所有人 |
| 首页 | 查看附近车位 + 我的车位 | 所有人 |
| 车位详情 | 查看单个车位信息、申请借用 | 借用方 |
| 发布共享 | 设置空闲时段和价格 | 出租方 |
| 借用记录 | 我的借用历史 | 所有人 |
| 个人设置 | 修改昵称、楼栋、常用区域 | 所有人 |
| 管理后台 | 审核、统计数据 | 管理员 |

- 首次登录 → 跳转"完善信息"页

## 2. 完善信息页（首次登录）

```
┌─────────────────────────┐
│      完善你的信息          │
│                         │
│  昵称：[________]         │
│                         │
│  楼栋：[▼ 选择楼栋]       │
│  单元：[▼ 选择单元]       │
│                         │
│  常用停车区域：            │
│  ┌─────┬─────┬─────┐    │
│  │ B1-A │ B1-B │ B1-C │    │
│  ├─────┼─────┼─────┤    │
│  │ B2-A │ B2-B │ B2-C │    │
│  └─────┴─────┴─────┘    │
│  （选一个你平时停车的区域）  │
│                         │
│  [ 完成 ]               │
└─────────────────────────┘
```

- 楼栋：1幢 ~ 30幢（手动维护列表）
- 单元：根据楼栋动态显示（如 20 幢有 1单元、2单元）
- 常用区域：用户自己选，后续首页按此区域优先展示

### 首页排序逻辑
1. 同区域车位排最前
2. 价格低的排前面
3. 空闲时间长的排前面

## 6. 管理后台（PC 端）
