# 页面设计

## 页面总览

| 页面 | 用途 | 谁用 |
|------|------|------|
| 登录页 | 手机号登录/注册 | 所有人 |
| 完善信息页 | 首次登录填写资料 | 新用户 |
| 首页 | 查看附近车位 + 我的车位 | 所有人 |
| 车位详情 | 查看单个车位信息、申请借用 | 借用方 |
| 发布共享 | 设置空闲时段和价格 | 出租方 |
| 借用确认页 | 确认借用 + 一键通知管家 | 出租方 |
| 借用记录 | 我的借用历史 | 所有人 |
| 管理后台 | 审核、统计数据 | 管理员 |

## 2. 完善信息页（首次登录）

```
┌─────────────────────────┐
│      完善你的信息          │
│                         │
│  昵称：[________]         │
│                         │
│  楼栋：[▼ 选择楼栋]       │
│  单元：[▼ 选择单元]       │
│                         │
│  常用停车区域：            │
│  ┌────┬────┬────┐       │
│  │  A  │  B  │  C  │       │
│  ├────┼────┼────┤       │
│  │  D  │  E  │  F  │       │
│  ├────┼────┼────┤       │
│  │  G  │     │     │       │
│  └────┴────┴────┘       │
│  （选一个你平时停车的区域）  │
│                         │
│  [ 完成 ]               │
└─────────────────────────┘
```

- 楼栋：1幢 ~ 30幢（从 buildings 表加载）
- 单元：根据楼栋动态显示（如 20 幢有 1单元、2单元）
- 常用区域：A~G（从 zones 表加载），用户自己选，首页按此优先展示

```
├─────────────────────────┤
│                         │
│  📍 附近可借车位（G区）   │
│  ┌─────────────────────┐│
│  │ G-012  4元/h  ⭐同区域 ││
│  │ G-015  5元/h  ⭐同区域 ││
│  │ A-003  4元/h  其他区域 ││
│  └─────────────────────┘│
│  [查看更多]              │
│                         │
│  ─────────────────────  │
│                         │
│  🚗 我的车位             │
│  ┌─────────────────────┐│
│  │ A-005  状态：空闲      ││
│  │       [发布共享]      ││
│  └─────────────────────┘│
│                         │
│  ─────────────────────  │
│                         │
│  📋 最近借用             │
│  ┌─────────────────────┐│
│  │ 无记录               ││
│  └─────────────────────┘│
│                         │
├─────────────────────────┤
│  [首页]  [发布]  [记录]  │
└─────────────────────────┘
```

### 首页排序逻辑
1. 同区域车位排最前（标 ⭐ 同区域）
2. 价格低的排前面
3. 其他区域排后面

```
│  ┌─────────────────────┐│

│  │                     ││
│  │ 您好，我是20幢2单元   ││
│  │ 的业主张三。          ││
│  │ 我的车位 A-005 已借  ││

│  │ 车牌号：浙A·12345    ││
│  │ 麻烦知悉，谢谢！      ││

│                         │
└─────────────────────────┘
```

- 点"复制文案" → 拷贝到剪贴板 → 粘贴发给物业管家
- 纯前端生成，无需后端改动

## 7. 管理后台（PC 端）

- 用户列表：查看所有用户、审核认证
- 车位列表：查看所有车位状态
- 数据统计：总用户数、总车位数、今日借用数
- 楼栋管理：维护楼栋-单元列表
- 区域管理：维护 A~G 区域数据
