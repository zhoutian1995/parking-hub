## 特殊车位

从 PDF 图片中可见：
- **无障碍车位**：蓝色轮椅符号标记
- **充电桩车位**：闪电符号标记
- 具体分布在哪些编号：待业主绑定时标注

## 布局结构（B2 为例）

- **通道**：灰色区域，双向箭头
- **单元分区**：标注为"1单元"和"2单元"
- **出入口**：蓝色门（楼梯间/电梯间）
- **非停车区**：粉色区域（储物间/配电间）
- **坡道**：左右两侧 V 形标记为上下坡道入口

## 数据录入计划

### 方式一：全部录入（推荐冷启动后）
SQL 批量生成 2,325 条 spot 记录，每条包含区域、编号、状态=空闲。

### 方式二：按需录入（MVP 阶段）
先录入 5~10 个车位用于冷启动，后续业主绑定车位时自动补入。
