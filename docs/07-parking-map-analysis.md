# PDF 车位图分析

## 概述

江南之星小区地下车库共两层，原始资料为 CAD 导出的 PDF 图纸（栅格图，非矢量）。

| 楼层 | 文件 | 说明 |
|------|------|------|
| 地下一层 | 地下一层车位图.pdf | 120×170cm 平面图 |
| 地下二层 | 江南之星地下室二层车位.pdf | 平面图 |

> ⚠️ **重要说明**：PDF 是 CAD 导出的栅格图（位图），AI 无法精确提取车位数据。
> 以下分析基于 AI 视觉识别 + 人工判断，标注了置信度。精确数据需人工现场核对或找物业要原始 CAD 文件。

## ⚠️ 需要人工确认的事项

| 事项 | 优先级 | 说明 |
|------|--------|------|
| B1 是否有 4#、6# 区域 | 高 | AI 两次分析矛盾 |
| 每个区域的精确车位数 | 高 | AI 只能估算，无法精确 |
| 车位编号格式（B1 和 B2） | 高 | 需要看图上的具体编号 |
| 特殊车位分布 | 中 | 无障碍、充电桩具体在哪 |
| B2 的 B1-B5 编号是否准确 | 中 | AI 识别可能有误 |

### 建议获取方式
1. **找物业要 CAD 原始文件**（最准确）
2. **人工对照 PDF 数车位**（次选）
3. **先录入 5-10 个车位冷启动**，后续边用边录

## "一键通知管家"功能

### 需求
当借用确认后，车位业主（出租方）可以一键生成文案，复制后发给物业管家。

### 生成文案模板

```
您好，我是{楼栋}{单元}的业主{昵称}。
我的车位 {车位号} 已借给邻居使用。
停车时间：{开始时间} ~ {结束时间}
车牌号：{车牌号}
麻烦知悉，谢谢！
```

### 示例

```
您好，我是20幢2单元的业主张三。
我的车位 E-005 已借给邻居使用。
停车时间：今天 14:00 ~ 18:00
车牌号：浙A·12345
麻烦知悉，谢谢！
```

### 实现方式
- 借用确认后，前端生成文案
- 出租方看到"复制通知管家"按钮
- 点击复制，粘贴发给管家（微信/短信）
- 无需后端改动，纯前端生成

### 数据来源
- 楼栋、单元：从用户信息取
- 昵称：从用户信息取
- 车位号、车牌号、时间：从借用记录取
