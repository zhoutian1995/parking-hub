## 验收标准

每个功能完成后，按以下标准自测：

| 维度 | 评分标准 |
|------|----------|
| 功能完整 | 能正常走通流程，无报错 |
| 用户体验 | 操作步骤 ≤ 3 步，加载 ≤ 2 秒 |
| 边界处理 | 空状态、错误提示、异常情况都有处理 |
| 数据正确 | 数据库记录正确，前端显示一致 |

## 实测通过的功能流程

1. **登录流程**：测试账号登录 → 自动跳转首页 ✅
2. **完善信息**：点击昵称 → 编辑楼栋/单元/区域/联系方式 → 保存 ✅
3. **绑定车位**：我的车位 → 绑定更多 → 输入车位号 → 提交 ✅
4. **发布车位**：选择车位 → 设置时段/价格/备注 → 确认发布 ✅
5. **搜索车位**：输入车位号 → 显示结果 → 点击查看详情 ✅
6. **申请借用**：查看详情 → 输入车牌号 → 申请使用 ✅
7. **结束借用**：最近借用 → 点击"结束" → 状态变为已完成 ✅
8. **通知管家**：点击"通知管家" → 生成文案 → 复制 ✅

## 待优化项

1. **短信通知**（P1）：用户借用/确认/结束时发送短信通知
2. **搜索结果交互**：搜索结果点击后弹出详情页的交互可以更明显
3. **借用流程完善**：确认/拒绝借用的具体交互流程（当前需车主端操作）
