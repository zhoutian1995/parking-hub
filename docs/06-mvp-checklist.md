## 阶段二：体验优化（用户量 50+ 后）

- SVG 车位地图（可视化展示）
- 搜索车位号
- 车位收藏
- 借用评价
- 微信模板消息通知

## 阶段三：高级功能（用户量 200+ 后）

- AI 自然语言搜索（"我要找 B2 区 4 块钱以下的车位"）
- 智能推荐（根据用户习惯推荐）
- 月卡/季卡模式
- 自动计费（接入停车场系统）

## 开发顺序（本周）

```
Day 1: 数据库迁移 + 楼栋/区域数据录入
Day 2: 后端 API（nearby + buildings + zones + me 更新）
Day 3: 前端首页重做
Day 4: 前端完善信息页 + 发布页
Day 5: 联调测试 + 部署上线
```

## 验收标准

每个功能完成后，按以下标准自测：

| 维度 | 评分标准 |
|------|----------|
| 功能完整 | 能正常走通流程，无报错 |
| 用户体验 | 操作步骤 ≤ 3 步，加载 ≤ 2 秒 |
| 边界处理 | 空状态、错误提示、异常情况都有处理 |
| 数据正确 | 数据库记录正确，前端显示一致 |
